# Assets Required

Place the following files in the `assets/` folder:

## Required Files:

1. **icon.png** (1024x1024px)
   - App icon
   - PNG format
   - Square, no transparency in corners

2. **splash.png** (1284x2778px)
   - Splash screen
   - PNG format
   - Black background recommended

3. **adaptive-icon.png** (1024x1024px)
   - Android adaptive icon
   - PNG format with transparency

4. **favicon.png** (48x48px)
   - Web favicon
   - PNG format

5. **notification-icon.png** (96x96px)
   - Notification icon
   - PNG format, white on transparent

6. **alarm.mp3** (Optional)
   - Custom alarm sound
   - MP3 format
   - If not provided, app uses online sound

## Quick Setup:

You can use placeholder images for now:
- Just create simple black squares with white text for testing
- Or download free icons from https://www.flaticon.com/

The app will work without these files initially, but you'll need them for building the standalone app.
